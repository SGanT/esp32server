# Простой файловый сервер

## Prerequirements

- Платы семейства ESP32
- Доступная flash память >= 4 mb (используется кастомная таблица разделов)

## Сборка
- Задайте настройки Wi-Fi в файле `wifi.h`, дальше как обычно с ESP-IDF

## Запуск

Как только бинарник вшит в контроллер, и контроллер подключился к Wi-Fi сети, можно обратиться в браузере по его IP адресу и открыть SPA приложение
